## Overview

SCIM Inspector is a desktop application for testing SCIM 2.0 provisioning endpoints. It provides:

- An interactive API explorer with pre-built SCIM operations
- Automated compliance validation across 7 test categories
- Load testing with real-time metrics
- Field mapping rules for custom validation
- Report export (JSON, CSV, PDF)

See [Getting Started](getting-started.md) to set up your first server connection.
